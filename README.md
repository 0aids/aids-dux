# aids.dux
Very slightly modified [Architeuthis.Dux](https://github.com/tapioki/cephalopoda/tree/main/Architeuthis%20dux).
Only major difference is a 3-key thumb cluster, and only the PCB for holding the keys.

Purposely designed in this weird way for prototyping purposes only.

*NOT IN FINISHED STATE* (Just ordered the parts)\
Planning to be partially hand-wired.
